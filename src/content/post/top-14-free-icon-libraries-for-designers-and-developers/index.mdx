除了框架本身，我在开发中最依赖的其实是一套“工具库组合拳”。以下 8 个开源项目分别解决了日志、校验、性能、对象拷贝等痛点，也是我在校园项目、实习任务中频繁引用的利器。

## 1. Hutool

集合工具、HTTP、Excel、ID 生成等常用能力。Spring Boot 项目里经常需要字符串处理、加解密、时间格式化，用 Hutool 可以省下大量样板代码。

## 2. MapStruct

DTO 与实体转换神器。以往手写 `BeanUtils.copyProperties` 既慢又容易漏字段，MapStruct 通过注解生成类型安全的 Mapper，并支持自定义表达式。

## 3. Sa-Token

轻量级权限框架，兼容不同认证模式。在竞赛管理系统和“趣游兔”两个项目里，我都基于 Sa-Token 实现了多角色、多租户的权限控制。

## 4. Resilience4j

熔断、限流、隔仓必备。结合 Spring Boot Actuator 能实时查看状态，配合 Prometheus 监控非常方便。

## 5. MyBatis-Plus

自动 CRUD、Wrapper 条件构造器、逻辑删除等特性显著提升开发效率。再配合租户插件就能够快速实现数据隔离。

## 6. EasyExcel

在成都火耕的实习中，我们需要导出百万级订单数据，EasyExcel 的分批写出和流式处理帮我轻松搞定。

## 7. Thumbnailator

处理图片的 Java 库，在 VR 模块中负责压缩、裁剪和生成多分辨率切片，搭配定时任务即可自动化处理大批量资源。

## 8. SpringDoc / Swagger

文档永远是沟通的桥梁。SpringDoc 能自动暴露 OpenAPI 文档，我们的 CI/CD 流水线会在测试环境部署后自动生成接口说明，方便前后端联调。

欢迎把你喜欢的 Java 库分享给我，大家一起扩展工具箱。
